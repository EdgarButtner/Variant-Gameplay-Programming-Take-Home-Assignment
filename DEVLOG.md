# Edgar's DevLog

---

**What I Built**

I built a musical typing game centered around the core given Spline character. I was first inspired by rhythm and beat games such as Magic Tiles and Friday Night Funkin' where the player tries to tap on the screen or press arrow keys on the beat of the song. Secondly I wanted to try and combine this with something like Typing.com where the player has to type words as quickly as possible. From these two inspirations I came up with Combo Hero! I decided I would play a song as the lyrics of said song came falling down from the top of the screen and have the player type them before they reached the bottom. I also hoped to implement a system where the player should be rewarded for typing on beat but did not have the time. I did end up implementing a scoring mechanic as well as levels which allow the player to earn a higher score. The level progresses naturally and gains extra progress when a word is typed correctly. If the user makes too many errors the game ends and transitions to a game over scene.

I also aimed to incorporate the Spine character by portraying them as a singer / dancer on a stage. My goal was to incorporate Spine animations by having my character dance or do an animation with every word typed giving the player a more intimate way of interacting with the game environment.

---

**My Technical Challenge**

I decided to challenge myself by using Javascript, which I had never used before! I have experience with TypeScript as well as Java so the learning curve was quite forgiving.

My main technical challenge did not come from having to learn a new language, however, but from trying to find out how scenes and scene keys linked together. Most of my game development background has been in engines such as Unity and Godot so learning how a web based game engine handled scenes and scene transitions was difficult but I did a deep dive into Phaser 4 documentation and eventually got the hang of it. Over all development was slightly challenging but very rewarding!

I did not run into too many troubling bugs or errors during this process but when I did I primarily used the console to debug my project.

---

**What I'd Add**

There were a couple things which I was not able to get to that I really wished I had time for. First of all I did not get to incorporate the actual lyrics of the song into the game. I was bound simply by time as it would have been an easy task. As of right now my game simply spawns a random word from the list of animations. Secondly I did not get to sync word spawning to the beat of the song which I wished I could have done. Finally I would have really liked to implement a concert stage themed background to really immerse the player in the world of my game!

Unfortunately I only had roughly 2 days to work on this assignment as the rest were taken up because of schoolwork and personal matters.
