# Edgar's DevLog

---

**What I Built**

I built a musical typing game centered around the core given Spline character. I was first inspired by rythm and beat games such as Magic Tiles and Friday Night Funkin' where the player tries to tap on the screen or press arrow keys on the beat of the song. Secondly I wanted to try and combine this with something like Typing.com where the player has to type words as quickly as possible.

I also aimed to incorporate the Spine character by portraying them as a singer / dancer on a stage. My goal was to incorporate Spine animations by having my character dance or do an animation with every word typed giving the player a more intimate way of interacting with the game enviroment.

---

**My Technical Challenge**

I decided to challenge myself by using Javascript, which I had never used before! I have experience with Typescipt as well as Java so the learning curve was quite forgiving.

My main technical challenge did not come from having to learn a new language, however, but from trying to find out how scenes and scene keys linked together. Most of my game development background has been in engines such as Unity and Godot so learning how a web based game engine handled scenes and scene transitions was difficult but I did a deep dive into Phaser 4 documentation and eventually got the hang of it. Over all development was slightly challenging but very rewarding!

---

**What I'd Add**

I did not get to sync word spawning to the beat of the song which I wished I could have done. Secondly I would have really liked to implement a concert stage themed background to really immerse the player in the world of my game!
